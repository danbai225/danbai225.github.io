golang 申请一块内存

摘要：这是一个用于申请一块占用实际内存的内存空间的函数。函数的参数是所需内存空间的大小（以字节为单位）。函数会使用`make`函数创建一个大小为`size`的字节切片。然后，通过一个循环，将切片中的每个元素赋予递增的值，以模拟对内存的占用。最后返回该字节切片作为申请到的内存空间。

申请一块占用实际内存的内存空间。

func RequestMemory(size int64) []byte {
	bytes := make([]byte, size)
	for i := range bytes {
		if i == 0 {
			bytes[i] = 0
		} else {
			bytes[i] = bytes[i-1] + 1
		}
	}
	return bytes
}
