# angular打包内存占用过高

通过如下命令限制
`node --max_old_space_size=600 ./node_modules/@angular/cli/bin/ng build`

# golang打包压缩体积

`go build -ldflags '-w -s' -o main`
`upx main`(需要先安装[upx](https://github.com/upx/upx))

# js提取页面文本内容

` document.documentElement.innerText`
